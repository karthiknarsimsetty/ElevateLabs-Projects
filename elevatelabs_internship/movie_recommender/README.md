# 🎬 CineMatch AI - Movie Recommendation System

A modern AI-powered Movie Recommendation System built using **Python, Streamlit, Machine Learning, and TF-IDF**. The application recommends similar movies based on movie content using cosine similarity.

---

## 🚀 Features

- Modern Streamlit UI
- Beautiful animated interface
- Search from 300+ movies
- English, Hindi, and Telugu movies
- AI-based recommendations
- Similarity percentage
- Genre display
- Movie ratings
- Responsive layout

---

## 🛠️ Technologies Used

- Python
- Streamlit
- Pandas
- Scikit-learn
- TF-IDF Vectorizer
- Cosine Similarity

## ⚙️ Installation

Clone repository

```bash
git clone https://github.com/yourusername/Movie-Recommendation-System.git
```

Move into folder

```bash
cd Movie-Recommendation-System
```

Install dependencies

```bash
pip install streamlit pandas scikit-learn
```

Run application

```bash
streamlit run app.py
```

---

## 🎯 How It Works

1. Select a movie.
2. The movie description is converted into TF-IDF vectors.
3. Cosine Similarity calculates similarity scores.
4. The most similar movies are recommended.
5. Results display:

- Match Percentage
- Rating
- Genre
- Language

---

## 🎥 Dataset

The project contains **300 movies**, including:

- 100 English Movies
- 100 Hindi Movies
- 100 Telugu Movies

Each movie contains:

- Title
- Genre
- Description
- Rating
